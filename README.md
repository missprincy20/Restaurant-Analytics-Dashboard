# 🍽️ Restaurant Analytics Dashboard

A professional **Restaurant Analytics Dashboard** built with **Python, Streamlit, Pandas, and Plotly**. This project provides interactive data visualization, restaurant exploration, recommendations, comparison, and geographic analysis using a restaurant dataset.

The application helps users discover restaurants, analyze ratings, compare restaurants, explore cuisines, and make better dining decisions through an easy-to-use web interface.

# Installation

Clone the repository

```bash
git clone https://github.com/missprincy20/Restaurant-Analytics-Dashboard.git
```

Move into the project folder

```bash
cd Restaurant-Analytics-Dashboard
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
streamlit run app.py
```
